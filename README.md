# Java-GUI-
📘 Java GUI Quiz Project
📝 Description

This is a simple Java GUI application built using Java Swing.
The program allows users to enter text, copy it, style it, change colors, and move it using keyboard arrows.

🚀 Features
✍️ Write text in a text field
📋 Copy text to a label
🔘 Enable / Disable input field
🎨 Change text color (Red / Green)
🔠 Text styling:
Bold
Italic
Plain
Underlined
💾 Save button with confirmation message
⌨️ Move text using arrow keys (↑ ↓ ← →)
🖼️ UI Components
JFrame → Main window
JTextField → Input text
JLabel → Display text
JButton → Copy & Save
JRadioButton → Enable/Disable
JCheckBox → Text styles
MouseListener → Color selection
KeyListener → Move label
🎮 Controls
Action	Function
Copy Button	Copies text to label
Enable	Enables typing
Disable	Disables typing
Red / Green	Change text color
Checkboxes	Change text style
Arrow Keys	Move text
Save	Show success message
🛠️ Technologies Used
Java
Swing (GUI)
AWT Event Handling
▶️ How to Run
Open the project in any Java IDE (like IntelliJ or Eclipse)
Compile the file:
javac quiz2.java
Run the program:
java quiz2
